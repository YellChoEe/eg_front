## Emotion Game

  #### 패키지 인스톨
  - `yarn install`
  
  #### 로컬 서버 실행
  - `yarn dev`

  #### 빌드 분석
  - `yarn build:analyze`

  #### Production 빌드 및 실행
  - `yarn start`
